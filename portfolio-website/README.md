# Personal Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and vanilla JavaScript. This website serves as a central hub to introduce myself and link to my various projects and interests.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Navigation**: Mobile-friendly navigation with smooth scrolling
- **Project Showcase**: Dedicated sections for games, writing, and other projects
- **Contact Integration**: Easy ways for visitors to connect with you
- **Performance Optimized**: Fast loading with optimized code

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About**: Personal information and skills showcase
3. **Projects**: Featured projects including games and writing
4. **Links**: Quick access to your various platforms and projects
5. **Contact**: Multiple ways for people to reach out
